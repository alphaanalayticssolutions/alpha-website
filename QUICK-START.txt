## 📞 WHAT TO DO IF SOMETHING'S WRONG

### Logo not showing?
- Check file exists at: `images/logo.png`
- Check file name is exact (case-sensitive)
- Clear browser cache and refresh

### Buttons not working?
- Check browser console for errors (F12)
- Verify `js/main.js` is loaded

### Styles not loading?
- Check `css/styles.css` file path
- Verify no typos in link tag
- Clear browser cache

### Mobile menu not opening?
- JavaScript may not be loading
- Check browser console (F12)
- Verify `js/main.js` is loaded
