## 🔧 CUSTOMIZATION GUIDE

### Change Colors:
Edit CSS file (`css/styles.css`) lines 8-14:
```css
:root {
    --red: #C1272D;        /* Change to your red */
    --blue: #1A5490;       /* Change to your blue */
    --white: #FFFFFF;
    --bg: #F8FAFB;
}
```

### Change Hero Background Image:
Edit CSS file line 177:
```css
background: url('YOUR-IMAGE-URL');
```

### Change Contact Section Background:
Edit CSS file line 628:
```css
background: url('YOUR-IMAGE-URL');
```

### Change Content:
Edit `index.html` - all text is clearly marked with comments

### Change Fonts:
Currently using:
- Headings: Playfair Display (serif)
- Body: Inter (sans-serif)

To change, update Google Fonts link in `index.html` head section.
