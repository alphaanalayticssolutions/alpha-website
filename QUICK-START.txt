## ✏️ Quick Edits

### Add Your Logo:
Replace the text logo in navbar:
```html
<div class="logo">
    <img src="images/logo.png" alt="Alpha Applied Analytics Solutions">
</div>
```

### Change Colors:
Edit `css/styles.css` at the top:
```css
:root {
    --color-blue: #2E5C8A;
    --color-red: #C1272D;
}
```

### Add Images:
1. Put images in `images/` folder
2. Reference in HTML:
   `<img src="images/your-photo.jpg" alt="Description">`
