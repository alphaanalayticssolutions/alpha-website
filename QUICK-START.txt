# QUICK START GUIDE
# Alpha Analytics Solutions Website

## 🚀 Getting Started in 3 Steps

### Step 1: Extract Files
Unzip the folder and you'll see:
- index.html (Services Page)
- workflow.html (AI Workflow Page)
- css/ folder (styles)
- js/ folder (interactivity)
- README.md (full documentation)

### Step 2: View Locally
Simply double-click `index.html` to open in your browser.
- Navigation works between both pages
- All features work locally (no server needed)

### Step 3: Deploy to Web
Upload all files to your web hosting:
- Netlify: Drag & drop the entire folder
- GitHub Pages: Push to repository
- Traditional hosting: FTP upload to public_html
